# DQN_Five_in_Line

### Deep Q-Networkを用いた五目並べ
#### コンセプト
* 対戦すればするほど強くなるエージェント。
* 対戦が終わると，Replay Bufferとして保存。学習に利用。

#### 現状
* とりあえず，5×5マスのテーブルでの三目並べを実装(学習まで実装済み)

#### 今後の予定
* 5×5マスのテーブルでは，CPUが勝つのは難しいのでテーブルを大きくする予定。
